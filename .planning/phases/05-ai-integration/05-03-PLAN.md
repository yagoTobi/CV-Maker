- Create `handleDownload` callback:
  ```typescript
  const handleDownload = useCallback(async () => {
    if (!formData) return;
    setIsDownloading(true);
    try {
      // Step 1: Generate LaTeX from formData
      const { texContent, error: genError } = await api.generateLatex(formData);
      if (!texContent || genError) {
        console.error('LaTeX generation failed:', genError);
        setIsDownloading(false);
        return;
      }
      // Step 2: Compile LaTeX to PDF
      const result = await api.compileLatex(texContent, formData.templateId);
      if (!result.success || !result.pdf_base64) {
        console.error('PDF compilation failed:', result.error);
        setIsDownloading(false);
        return;
      }
      // Step 3: Trigger browser download
      const byteChars = atob(result.pdf_base64);
      const byteArray = new Uint8Array(byteChars.length);
      for (let i = 0; i < byteChars.length; i++) {
        byteArray[i] = byteChars.charCodeAt(i);
      }
      const blob = new Blob([byteArray], { type: 'application/pdf' });
      const url = URL.createObjectURL(blob);
      const a = document.createElement('a');
      a.href = url;
      a.download = cvFilename(formData);
      document.body.appendChild(a);
      a.click();
      document.body.removeChild(a);
      URL.revokeObjectURL(url);
    } catch (err) {
      console.error('Download failed:', err);
    }
    setIsDownloading(false);
  }, [formData]);
  ```
- Import `api` from `../../services/api` (already imported? check -- it may need to be added).

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| File picker -> useImport -> backend | User-selected file uploaded to /cv-import endpoint |
| Backend response -> setFormData | AI-extracted CVFormData loaded into editor state |
| formData -> generateLatex -> compileLatex | Form data sent to backend for LaTeX generation and PDF compilation |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-07 | Tampering | Import formData injection | mitigate | Imported formData passes through existing backend validation (Pydantic models on /cv-import). Frontend sets templateId explicitly (not from import). setFormData replaces entire state -- no partial injection. |
| T-05-08 | Denial of Service | Large file upload | accept | Backend already enforces MAX_FILE_SIZE = 10MB. Frontend file picker accepts only .pdf/.docx/.json. Existing mitigation sufficient. |
| T-05-09 | Information Disclosure | PDF download blob URL | mitigate | URL.revokeObjectURL called immediately after download link click, preventing URL reuse. PDF contains only user's own CV data. |
| T-05-10 | Tampering | LaTeX injection via formData | accept | Backend latex_compiler.py already sanitizes LaTeX content via _sanitize_content(). No new attack surface. |
</threat_model>
