Create useScrollSync.ts:

```typescript
import { useEffect, useRef, type RefObject } from 'react';

/**
 * useScrollSync -- Watches visible CV sections via IntersectionObserver
 * and scrolls the ChangePanel to align cards with the topmost visible section.
 *
 * Direction: CV scroll drives panel scroll (one-way). Panel scroll does NOT
 * drive CV scroll.
 *
 * Anti-jitter: isAutoScrolling ref flag prevents observer -> scroll -> observer loops.
 */
export function useScrollSync(
  cvContainerRef: RefObject<HTMLElement | null>,
  panelRef: RefObject<HTMLElement | null>,
  enabled: boolean
): void;
```

Implementation:
1. `useEffect` that runs when `enabled`, `cvContainerRef`, `panelRef` change.
2. Guard: if `!enabled || !cvContainerRef.current || !panelRef.current` return early.
3. Create `isAutoScrolling` ref (useRef<boolean>(false) declared at hook top level).
4. Query `cvContainerRef.current.querySelectorAll('[data-section]')` to get all section DOM elements.
5. Create `IntersectionObserver` with options: `{ root: null, threshold: 0.3 }`.
6. Observer callback:
   - If `isAutoScrolling.current` is true, skip (return early).
   - Filter entries to those with `isIntersecting === true`.
   - Sort by `boundingClientRect.top` ascending to find the topmost visible section.
   - If none visible, return.
